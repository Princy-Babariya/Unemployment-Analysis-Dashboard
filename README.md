# 📊 Unemployment Analysis Dashboard

This project is an interactive dashboard built using Python to analyze unemployment trends and patterns.

## 🚀 Features
- Trend analysis of unemployment over time
- Seasonal pattern analysis
- State-wise comparison
- COVID-19 impact visualization
- Forecasting future unemployment trends
- Interactive filters (state & year)

## 🛠️ Tech Stack
- Python
- Pandas
- Flask
- Data Visualization

## 📌 Key Insights
- Unemployment spiked during COVID-19
- Seasonal patterns observed across months
- Certain states show consistently higher unemployment rates

## 📂 How to Run
1. Clone the repository  
2. Install dependencies  
3. Run `app.py`  
4. Open in browser
